# YunYuanshen-Wish-Export
云原神抽卡记录导出器，4.6更新后的替代版本，无需扫码，通过云游戏获取抽卡地址

我自己用的2个虚拟机和3台真机都能够成功运行，但是确实遇见过部分人的机器无法运行的。可以尝试在云游戏登录界面打开“兼容模式”。
如果有解决方法，可以pull request。也可以issue提出。
